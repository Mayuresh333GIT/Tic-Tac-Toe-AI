# Tic-Tac-Toe-AI
This is a game created by me in form of a project for college. In this game the computer plays against you and uses the minimax algorithm to be unbeatable. 

**What the project does:** This project when run properly will show the user a Tic Tac Toe game with a simple user interface which is made using the 'tkinter' library in python. This game is built using minimax algorithm along with alpha beta pruning algorithm to make optimal decisions. The AI evaluates all possible game states and chooses the move that maximizes its winning chance while minimizing the player's chances of winning. Because of this strategy, the AI is difficult to defeat.
The graphical interface has features where players with the board, select their symbol, and restart the game when needed.

**Technologies Used**:
1.Python
2.Tkinter (for GUI)
3.Minimax Algorithm
4.Alpha-Beta Pruning

**How to set it up:** The user can set the project to test for themselves by following these steps:
1. It is advised that one should use VS Code for viewing this project
2. The user should copy the project codes from the file in the repository and should run it by clicking the run icon in the VS Code after pasting it.

**How to use it:**
1. Run the Python file as instructed in the VS Code
2. Choose whether you want to play as X or O.
3. Click on any square on the grid to place your symbol.
4. The AI will automatically make its move.
5. The game continues until: A player wins or the game ends in a draw.
6. The winner or result will be displayed at the top.
7. Click the Restart button to play again.
